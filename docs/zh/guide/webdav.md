# WebDAV

## WebDAV 配置

| Name     | Value                       |
| -------- | --------------------------- |
| Url      | http[s]\://domain:port/dav/ |
| Host     | domain                      |
| 路径     | dav                         |
| 协议     | http/https                  |
| 端口     | 与网页端一致                 |
| WebDAV用户名 | 与网页端用户名一致          |
| WebDAV密码 | 与网页端密码一致            |
