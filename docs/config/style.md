# 样式设置

## logo

徽标显示在页面的左上角。 默认为 AList Logo

## 网站图标

favicon 显示在浏览器选项卡中。 默认为 AList Logo

## 主色调

全站页面的主要颜色。默认值为 `#1890ff`。

## 主页图标

主页图标显示在导航栏的左侧。 默认是 `🏠`。

## Hope容器

- Max 980px：主宽度将设置为 `min(99%, 980px)`
- Hope容器：设置内容的最大宽度以匹配当前断点的最小宽度，参考 https://hope-ui.com/docs/layout/container。
